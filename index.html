Git starter commands

Starter commands

git init Initialise un repository local

git config --local user.name <user name> Initialise nom utilisateur (ajoute dans le fichier de config)

git config --local user.email <user email> Initialise email utilisateur (ajoute dans le fichier de config)

Add

git add &mt;filename>Ajoute un fichier dans l'index

git add .Ajoute les fichiers du dossier en cours dans l'index

git add -AAjoute tous les fichiers dans l'index

Commit

git commit -m "your message"Commit des fichiers de la zone staging vers le repository

git commit -aCommit des fichiers de la zone staging vers le repository et ouvre l'editeur par defaut pour editer le message

Redact commit message

  • Prendre le temps de rédiger son message
  • Ajouter un sujet
  • Limiter le message à 50 caracteres
  • Ecrire sans accent
  • Répondre à la question "pourquoi ?" plutôt que "comment ?"

Affichage log

git log affiche les commits

git log -1 affiche le dernier commit

git log --pretty=oneline --max-count=2 affiche sur une ligne les deux derniers commits

git log --pretty=oneline --since="5 minutes ago" affiche les derniers commits dans les 5 dernieres minutes

git log --oneline --all --pretty --since="5 minutes ago" affiche les derniers commits dans les 5 dernieres minutes

Format de log

  • --pretty="..." format de sortie
  • %h abbreviation hash code commit
  • %d commit decoration
  • %ad commit date
  • %s commentaires
  • %an author name
  • --graph commit tree
  • --date=...

Version...

git tag affichage des release

git tag -a "your message" -m creation d'une release

Branche

git branch <branchName> Creation d'une branche

git checkout <branchName> Creation et connexion à une branche

git branch -d ou -D <branchName> Suppression d'une branche -d analyse des commits de la branche et -D force delete

git branch Liste des branches

git merge <branchName> Fusion de branches

Repository distant

git remote origin <url repository> Connexion à un repository distant

git push origin Transmet les commits locaux au repository distant d'une ou plusieurs branches

git fetch Importer des elements du repository distant

git clone <url repository> copie d'un projet dans un repository local
